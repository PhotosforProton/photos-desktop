How this list was produced
------------------------------------------------------------------------------

Nothing here is written from memory or inferred from a package name.

  npm packages   The sidecar bundle is rebuilt with esbuild's metafile enabled,
                 which reports every input file that was inlined. Each input is
                 mapped back to its package, and the licence is read from that
                 package's own package.json. Packages shipped unbundled beside
                 the sidecar are read directly from sidecar/build/node_modules.

  Rust crates    `cargo metadata --filter-platform x86_64-pc-windows-msvc`, for
                 both app/src-tauri and setup/src-tauri, walking normal
                 dependencies only from the workspace root. Build-dependencies
                 and dev-dependencies are excluded: neither is linked into a
                 shipped binary. Licence identifiers are the `license` field of
                 each crate's own Cargo.toml.

  Licence texts  Quoted from a real licence file on disk, with the file that was
                 quoted named at the top of each text.

A package whose licence could not be established from its own metadata would be
listed here as undetermined. At the time of writing there are none.

==============================================================================
1. Node.js runtime
==============================================================================

The application runs its sidecar on a copy of the official Node.js runtime,
installed as resources\sidecar.exe. It is an unmodified copy of node.exe as
distributed by the Node.js project, renamed. No Node.js source is modified.

  Node.js v24.15.0
  https://nodejs.org/
  Licence: MIT

The Node.js LICENSE file carries, in addition to the MIT licence covering
Node.js itself, the notices of the components Node.js bundles, among them V8,
OpenSSL, ICU, zlib and llhttp. Rather than reproduce all of it here, that file
is installed in full alongside this one as NODE-LICENSE.txt, taken verbatim
from the tag matching the version shipped:

  https://github.com/nodejs/node/blob/v24.15.0/LICENSE
